Bootanimations

Bootanimations must be created as Tagged Image File Format (.tiff) Sequences.

General

Folder structure:

  • Animation_Name/
    • b0/
      • 0000.tiff
      • 0001.tiff
      • ...
      • 9999.tiff
    • b1/
      • 0000.tiff
      • 0001.tiff
      • ...
      • 9999.tiff
    • b2/
      • 0000.tiff
      • 0001.tiff
      • ...
      • 9999.tiff
    • b.config

Everything in Animation_Name/b0/ will be executed once first, then the sequence in Animation_Name/b1/ will be looped until booting is finished. Finally Animation_Name/b2/ will be executed once to finish the animation.
The following Folder structure is also possible, but then only MODE=2 (see b.config section) will be the execution standart.

  • Animation_Name/
    • b0/
      • 0000.tiff
      • 0001.tiff
      • ...
      • 9999.tiff
    • b.config

b.config File

As discussed in General, default execution is selected. Alternative executions can be defined in b.config.

  • MODE = 0 - Default execution
  • MODE = 1 - b0 then b1 once and then b2 (everything once)
  • MODE = 2 - Loop over everything
